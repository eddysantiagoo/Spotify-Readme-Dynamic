What is Spotify Widget?

Spotify widget is a SVG template made with jinja2 and Python, with this project we consume the API of Spotify to show the last played song and the currently song that a user on Spotify is listening, this widget is full responsive and can be used in your README.md in GitHub or in any other website that you need.

How to use it?

For use this project, you need to have a Spotify Developer account (What is free) and clone or fork the repository. After that, only follow the tutorial of the README file to create your own widget and this is all, if you have problems, you can create a issue in the repo ðŸ’š
